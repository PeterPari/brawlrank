One tier list.
Nine sources.

Every week, pro players and creators publish conflicting tier lists — same brawler, two tiers apart, both partly right. BrawlRank pulls from 9 independent sources and combines them into one score, weighted by data quality and how recent each rating is.

View the Tier List

Base weights reflect data quality — win-rate sources count more, editorial opinions count less. All weights shrink as sources get older.

Tiers map to numbers (S = 6, A = 5, B = 4, C = 3, D = 2, F = 1). BrawlRank averages them, weighted by source quality and age. Past 15 days old, each source's weight halves every 7 days.

Update schedule

BrawlRank updates after major patches and ranked season resets, when the sources it tracks publish new data. Check the footer for the version and date.

Every update logs which brawlers moved tiers since the previous version.
