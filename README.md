## 📈 How It Works

BrawlRank uses a custom weighted scoring algorithm. We pull in tier rankings from various trusted sources within the Brawl Stars community. To make sure our data is completely relevant:
1. **Source Weighting:** Highly reputable and competitive sources carry more weight.
2. **Recency Decay:** A tier list from yesterday is more relevant than a tier list from two weeks ago. Older data naturally decays in value so the aggregated score accurately highlights the *current* meta.
